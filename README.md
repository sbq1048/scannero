## 不用装App的手机定位,听起来有点玄?

这个服务叫Scannero，听名字还以为是扫描仪什么的。但其实它是一个基于网页的手机定位服务,最大的特点就是——完全不用在对方手机上装任何东西。

你可能会说："扯淡吧,不装App怎么定位?"

这就是它巧妙的地方了。Scannero的工作原理其实很简单：你输入对方手机号，系统会自动发送一条带追踪链接的短信。对方一点开这个链接,GPS位置就会显示在你的地图上。整个过程完全基于浏览器,不需要下载任何软件。

说白了，就是通过GPS和基站数据来确定手机位置。这技术本身不新鲜,iPhone和Android都有自带的"查找"功能。但Scannero把这事儿做得更灵活了——你不需要提前设置什么家庭共享,也不用对方同意添加好友。只要有手机号,就能试试。

## 它到底有什么用?

### 1. 号码定位追踪

最基础的功能。你输入一个手机号，选择或自定义一条短信(比如"这是你的快递单号"、"点击查看照片"之类的),然后发送。对方一旦点击链接并同意分享位置,你就能看到具体的GPS坐标和地址。

不过这里要说清楚：如果对方不点,或者直接拒绝了位置请求,那就什么都看不到。所以这个功能的成功率,其实很依赖对方会不会点开那条短信。

### 2. 链接追踪

这个功能更灵活一点。你生成一个追踪链接,然后通过WhatsApp、微信、邮件、或者任何通讯工具发过去。对方一点开,位置就出来了。

如果你只知道对方的社交账号,没有手机号,这个方式就挺有用的。

### 3. 号码反查

接到陌生来电?输入号码,Scannero会告诉你这个号码的持有人信息、运营商、大致位置,还有垃圾评分。评分高的话,基本可以确定是推销电话了。

这个功能其实挺实用的。现在骚扰电话那么多,能提前知道对方身份,至少能少接几个保险推销。

### 4. 用户名反查

在社交网络或约会软件上认识了新朋友?输入对方的用户名,Scannero会帮你找到这个人在其他平台的账号,包括真名、关联的社交媒体等等。

网上交友嘛,小心一点总没错。

### 5. 找回丢失的手机

手机丢了,系统自带的"查找"功能用不了怎么办?

Scannero有个"找回手机"功能：你输入自己的手机号,写一条给捡到手机的人的信息(比如"如果你捡到这部手机,请点击这里联系我,我会给你报酬"),然后发送。只要对方点开链接,你就能看到手机的位置。

这招的前提是,捡到手机的人得愿意看短信,也得愿意点开。但总比完全没办法强。

### 6. 数据泄露检测

这个功能跟定位没关系,但挺有用的。输入你的邮箱或手机号,Scannero会扫描各种数据库,告诉你的信息有没有在数据泄露事件中被曝光。

现在各种数据泄露新闻那么多,自己的信息有没有被卖,确实值得查一查。

### 7. 匿名语音留言

这个功能有点神奇。你打一段文字,Scannero会把它转成语音,然后你可以匿名发给任何人。

说实话,我不太确定什么场景会用到这个。但既然做了,想必有它的道理。

## 合法性和隐私问题

这个问题很多人都会问。用这种服务追踪别人,合法吗?

首先，Scannero本身的设计是需要对方同意的。你发送的链接会明确请求位置权限,对方可以选择同意或拒绝。从这个角度说,它不算"偷偷摸摸"的监控。

但是，如果你通过伪装短信内容(比如假装是快递通知)来诱导对方点击,这个就进入灰色地带了。法律上怎么判,要看具体情况和当地法规。

一般来说:

- **追踪自己的未成年子女**：合法,这属于家长的监护权
- **追踪自己的设备**：合法,你有权追踪自己的财产
- **追踪配偶**：复杂,最好是在双方知情同意的情况下使用
- **追踪陌生人或没有正当理由的对象**：很可能违法

我的建议是:如果你要用这个服务,最好是在对方知情的情况下使用,或者至少要有正当理由(比如照看未成年子女、寻找丢失的手机等)。不要用它去干什么违法乱纪的事儿。
